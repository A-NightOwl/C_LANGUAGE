判断一个5位数是不是回文数。例如12321是一个回文数，即它的个位与万位相同，十位与千位相同。。

**输入格式:**
直接输入一个5位数正整数。

**输出格式:**
输出`yes`或`no`。此处均为小写字母

**输入样例:**

`12521`

**输出样例**:

`yes`

**输入样例:**

`65432`

**输出样例:**

`no`

```c
#include<stdio.h>
int main()
{
    int a,b,c,d,e;
    scanf("%1d%1d%1d%1d%1d",&a,&b,&c,&d,&e);
    if(a==e && b==d)
        printf("yes");
    else
        printf("no");
}
```
